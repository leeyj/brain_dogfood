효율적인 지식 관리와 탐색을 위한 모든 기능을 확인해보세요.

메모 작성

Editor
메모 서버는 슬러시(/) 명령어 체계를 지원하여 빠른 작업이 가능합니다.
Toast Editor를 통해 마크다운(Markdown)과 위지윅(WYSIWYG) 환경을 모두 제공하여 사용자 취향에 맞게 지식을 기록할 수 있습니다.

지식 분류 및 관리

Grouping

지능형 그룹 지정

에디터 창에서 $ 기호를 사용하여 즉시 그룹을 지정할 수 있습니다. 각 메모는 고유한 하나의 그룹을 가지며, 지정하지 않을 시 'default'로 설정됩니다.
Tagging

멀티 태그 시스템

입력 창에 태그를 직접 입력하거나, 본문 내에서 #태그 형식을 사용하여 마우스 동선 없이 빠르게 분류 체계를 구축할 수 있습니다.

기한(Due Date) 관리

중요한 일정이나 기한이 있는 메모는 기한을 설정하세요. 기한이 임박할수록 메모 카드의 테두리 색상이 변경되어 직관적으로 우선순위를 파악할 수 있습니다.
● 4일 이상 (안전)
● 2~3일 (주의)
● 당일 (긴급)

지식 보안 (AES-128)

민감한 정보는 AES-128 암호화를 적용하세요. 8자리 이상의 암호 설정 시 강력한 보안을 보장하며, 첨부파일까지 모두 암호화되어 안전하게 저장됩니다.
- 복호화: 메모리 상에서만 일시적으로 해제되며, 세션 종료나 새로고침 시 자동으로 다시 암호화됩니다.
- 영구 해제: 암호화 해제 후 명시적으로 저장해야 합니다.
주의: 암호를 분실할 경우 개발자도 복구가 불가능하므로 반드시 기억해 두시기 바랍니다.

지식 네비게이션 & 시각화

Weekly Calendar

주간 달력 및 히트맵

Alt+W 단축키를 통해 주 단위 지식 생산 빈도를 확인할 수 있습니다. 요일 옆의 점(Dot) 색상으로 기한 임박 메모를 즉시 파악할 수 있습니다.
Virtual Linker

버추얼 링커

Alt + 클릭을 통해 메모 간의 관계를 직관적으로 연결합니다. [[#id]] 형식을 통한 수동 링크도 지원합니다.
Link View

링크 뷰어

Ctrl + Alt + 클릭으로 특정 메모와 연결된 주체 및 참조자 관계를 파란색과 노란색으로 구분하여 시각화합니다.
Nebula Map

지식맵 (네뷸라)

작성된 모든 메모를 우주 성간처럼 시각화합니다. 그룹별 클러스터와 링크 관계(상호/일방 참조)를 한눈에 조망할 수 있습니다.

지식 탐색 및 관리

Explorer

지식 탐색기 (필터)

그룹과 태그별로 자동 분류된 지식 탐색기입니다. 화면 내에서 자유롭게 이동(Hover) 가능하며, 클릭 한 번으로 원하는 지식을 빠르게 필터링할 수 있습니다.
