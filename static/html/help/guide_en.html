Knowledge Classification

Grouping

Intelligent Grouping

Use the $ symbol in the editor to assign groups instantly. Each memo has a unique group; if unspecified, it defaults to 'default'.
Tagging

Multi-Tag System

Enter tags directly in the input box or use #tags within the text to build a taxonomy without moving your mouse.

Due Date Management

Set deadlines for important tasks. As the deadline approaches, the border color of the memo card changes to help you prioritize at a glance.
● 4+ Days (Safe)
● 2-3 Days (Warning)
● Today (Urgent)

Knowledge Security (AES-128)

Apply AES-128 encryption for sensitive information. A password of 8+ characters ensures strong security, including encrypted attachments.
- Decryption: Temporary unlock in memory; auto-relocks on session end or refresh.
- Permanent Unlock: Must be explicitly saved after decryption.
Caution: If the password is lost, data recovery is impossible even for developers.

Navigation & Visualization

Weekly Calendar

Weekly View & Heatmap

Toggle with Alt+W to track knowledge production. Dots next to weekdays indicate upcoming deadlines.
Virtual Linker

Virtual Linker

Use Alt + Click to link memos intuitively. Manual links like [[#id]] are also supported.
Link View

Link Viewer

Ctrl + Alt + Click to visualize relationships between source and reference memos in blue and yellow.
Nebula Map

Knowledge Map (Nebula)

Visualize all memos like a star cluster. Overview group clusters and link relationships (mutual/one-way references) at a glance.

Exploration & Management

Explorer

Knowledge Explorer (Filter)

Automatically categorized explorer by groups and tags. Freely draggable (Hover) and allows quick filtering with a single click.
