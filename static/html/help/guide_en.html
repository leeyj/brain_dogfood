📖 Basic Guide

Learn how to use the Knowledge Vault system efficiently.

1. Writing Memos

Type in the composer at the bottom and press Ctrl+Enter to save.

2. Using Tags

Include #tags in your content or use the tag input field.

3. Encrypted Knowledge

Click the lock icon to set a password for sensitive information.

4. Attachments

Drag and drop files or use the attach button to save multimedia.
