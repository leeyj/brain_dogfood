🚀 전역 네비게이션

Alt + 1 모든 메모 (All)
Alt + 2 고정 메모 (Pinned)
Alt + 3 오늘의 메모 (Today)
Alt + 4 이번주 메모 (Weekly)
Alt + 5 파일 모음 (Files)
Alt + 6 완료 및 보관 (Done)
Alt + 7 지식 탐색기 (Explorer)
Alt + 8 지식 맵 보기 (Nebula)
Alt + 9 환경 설정 (Settings)

✍️ 편집 및 동작

Ctrl + Enter 저장 및 게시 (Sync)
Ctrl + Alt + 1~4 카테고리 슬롯 지정
/ 🪄 슬래시 명령어
e ⚡ 즉시 수정 (마우스 호버)
Alt + Click 🔗 비주얼 노드 연결
ESC 🚪 모든 창 닫기
? ⌨️ 도움말 모달 열기
