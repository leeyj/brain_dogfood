## 3. 세부 기술 전략

### ① 암호화 메모 보안 강화
- **메모리 보호**: 복호화된 텍스트는 절대 파일 시스템에 쓰지 않으며, 오직 메모리 상의 변수로만 존재하다가 팝업을 닫으면 소멸됩니다.
- **오프라인 대응**: 서버 연결 없이도 로컬에 저장된 암호문을 플러그인 내장 로직으로 해독합니다.

### ② 일괄 전송(Bulk Push)과 충돌 방지
- **Selective Sync**: 사용자의 선택이 개입되는 UI를 제공하여 타 동기화 도구와의 레이스 컨디션을 방지합니다.
