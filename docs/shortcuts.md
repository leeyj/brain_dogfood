# ⌨️ 단축키 가이드 (Keyboard Shortcuts)

'뇌사료'를 더욱 빠르고 효율적으로 활용하기 위한 단축키 일람입니다. 
**Ctrl + Shift**를 주력으로 하며, 전설적인 **Quake 스타일**의 새 메모 단축키를 추가 지원합니다.

## 🚀 전역 단축키 (Global Navigation)
어느 화면에서나 즉시 실행됩니다.

- **`Alt + 1` ~ `9`**: 🚀 **왼손의 법칙 전역 네비게이션**
    - `Alt + 1`: 🌐 전체 지식
    - `Alt + 2`: 📌 고정 메모
    - `Alt + 3`: 📅 오늘의 메모
    - `Alt + 4`: 🗓️ 이번주 메모
    - `Alt + 5`: 📎 첨부파일 모음
    - `Alt + 6`: ✅ 완료 및 보관
    - `Alt + 7`: 🔍 지식 탐색기 (Explorer)
    - `Alt + 8`: 🕸️ 지식 네뷸라 (Graph)
    - `Alt + 9`: ⚙️ 환경 설정
- **`Alt + `** (Backtick): ⚡ **Quake 스타일 새 메모**
- **`Alt + W`**: 📅 **주간 뷰 토글**
- **`Alt + N`**: 📝 **새 메모 작성**
- **`e` (Mouse Over)**: ⚡ **즉시 수정** (메모 카드 위에서)
- **`?`**: ⌨️ **단축키 가이드 열기**

---

## ✍️ 메모 편집 (Editor Mode)
- **`Ctrl + Enter`** 또는 **`Ctrl + S`**: 💾 **저장 및 게시**
- **`Ctrl + Alt + 1 ~ 4`**: 🏷️ **카테고리 슬롯 즉시 지정**
- **`/` (Slash)**: 🪄 **슬래시 명령**
- **`Shift + ESC`**: 🗑️ **작성 취소**

---

## 🖱️ 마우스 인터렉션
- **#ID 클릭**: 📋 **링크 복사** (`[[#ID]]`)
- **Alt + #ID 클릭**: 🔗 **비주얼 노드 링커** (시각적 연결 시작)
- **카드 드래그 앤 드롭**: 📥 **링크 삽입** (작성기 위로 드래그 시)
- **클릭**: 상세 보기

---

> [!TIP]
> - **`Alt + `** 단축키는 게임 'Quake'의 콘솔창 호출 방식에서 유래한 것으로, 영감이 떠오른 순간 가장 빠르게 기록을 시작할 수 있는 방법입니다.
