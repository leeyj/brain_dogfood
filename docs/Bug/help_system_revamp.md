# 도움말 시스템 프리미엄 개편 보고서

## 변경 내용
- 원본 XML(`도움말_원본.xml`)의 방대한 가이드 내용을 바탕으로 기본 가이드(`guide_ko.html`)를 전면 개편.
- 총 16개의 기능 가이드 이미지를 추출하여 `static/img/help/`에 배치.
- 글래스모피즘 디자인과 애니메이션 효과를 적용하여 사용자 인터페이스(UI)를 현대적으로 개선.

## 조치 사항
1. **데이터 추출**: Python 스크립트를 사용하여 Word XML 형식에서 텍스트와 바이너리 이미지를 성공적으로 분리.
2. **콘텐츠 구조화**: 긴 텍스트를 섹션별 카드 레이아웃으로 분리하여 가독성 확보.
3. **캐시 관리**: `GuideContent.js`의 호출 버전을 `v3.0`으로 업데이트하여 변경된 디자인이 즉시 반영되도록 조치.

## 향후 주의 사항
- 새로운 기능 추가 시 원본 워드 문서를 업데이트하고, 동일한 추출 프로세스를 통해 도움말을 동기화할 것을 권장.
- 이미지 파일명이 `image1.png` 등 일련번호로 되어 있으므로, 유지보수 시 의미 있는 파일명으로 변경하는 것을 검토할 수 있음.
