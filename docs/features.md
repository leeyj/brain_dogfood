# 💎 핵심 기능 가이드 (v5.0+)

본 문서는 `뇌사료` 프로젝트를 상징하는 핵심 기능들인 **지식 시각화**, **주간 관리**, **암호화**, **AI 분석**에 대한 상세 명세를 담고 있습니다.

## 📅 1. 주간 지식 복기 (Weekly Knowledge Review) - v5.0 신규
단순한 기록을 넘어, 사용자가 주기적으로 자신의 지식을 되돌아볼 수 있도록 돕는 실무 지향적 인터페이스입니다.
- **주간 선택기**: 검색창 하단에 일~토 배지가 배치되어 이번 주의 활동을 한눈에 파악할 수 있습니다.
- **스마트 토글**: 📅 버튼 하나로 [주간 모아보기]와 [전체 지식 보기]를 즉시 전환하여 마우스 동선을 최소화합니다.
- **범위 필터링**: 개별 날짜 클릭 시 해당일의 메모를, 헤더 클릭 시 일주일 전체의 메모를 모아볼 수 있습니다.

## 🗒️ 2. 고밀도 리스트 레이아웃 (List Layout) - v5.0 신규
대량의 메모를 제목 위주로 빠르게 탐색해야 할 때 최적화된 보기 모드입니다.
- **가로형 카드**: 제목, 태그, 날짜, ID를 한 줄에 콤팩트하게 배치하여 수직 공간을 절약합니다.
- **즉시 조작**: 마우스 호버 시에만 나타나는 액션 버튼과 시각적 와이어링 기능을 통해 고밀도 환경에서도 직관적인 조작이 가능합니다.

## 🌌 3. 지식 네뷸라 (Knowledge Nebula)
D3.js v7 물리 시뮬레이션 엔진을 통해 파편화된 메모들을 유기적인 우주 성단 구조로 시각화합니다.
- **성단(Constellation) 로직**: 같은 그룹이나 공통 태그를 공유하는 노드들 사이에 인력을 설정하여 맥락이 유사한 지식들이 근접하게 배치됩니다.
- **인터랙션**: 마우스 호버 시 연결된 지식망이 강조(Highlight)되며, 실시간으로 지식 간의 위계를 파악할 수 있습니다.

## 🔗 4. 내부 링크 및 비주얼 노드 링커 (`[[#ID]]`)
- **비주얼 노드 링커**: 메모 카드의 #ID를 `Alt + 클릭` 드래그하여 지식과 지식을 선으로 잇는 '와이어링'을 수행합니다. 지식의 맥락을 설계하는 가장 직관적인 UX를 제공합니다.
- **드래그 앤 드롭**: 메모 카드를 작성기(Composer)로 드래그하여 참조 링크를 즉시 삽입합니다.

## 🔒 5. 이중 보안 암호화 시스템 (Dual-Layer Security)
- **개별 암호화**: 메모마다 고유한 비밀번호를 사용하여 `AES-256` 기반의 암호화를 제공합니다.
- **보안 마스킹**: 비밀번호 입력 및 리스트 뷰 노출 시 민감한 정보를 아스테리크(`*`)로 마스킹 처리하여 오프라인 환경에서의 숄더 서핑(Shoulder Surfing)을 방지합니다.

## 🧠 6. Gemini AI 기반 지식 구조화 (AI Insight)
- **자동 추출 및 요약**: Gemini 2.0 모델이 메모의 맥락을 분석하여 핵심 요약과 태그를 생성합니다.
- **지능형 연동**: AI가 생성한 태그는 지식 네뷸라 엔진의 인력 설정에 반영되어 자동으로 지식의 지도를 형성합니다.

## ⌨️ 7. 고속 워크플로우 (Productivity)
- **즉시 수정 (Instant Edit)**: 메모 카드 위에 마우스를 올리고 `e` 키를 누르면 즉시 편집 모드로 진입합니다.
- **슬래시 명령**: `/task`, `/ai`, `/h2` 등 풍부한 명령어로 서식 지정을 가속화합니다.
