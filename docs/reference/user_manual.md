# 📔 뇌사료 (Brain Dogfood) 사용자 매뉴얼 (v8.1+)

'뇌사료' 프로젝트에 오신 것을 환영합니다! 본 매뉴얼은 파편화된 영감을 체계적인 지식 성단(Nebula)으로 구축하고, 주간 단위로 복기하며 성장하는 데 필요한 모든 가이드를 제공합니다.

---

## 🎨 1. 새로운 시각적 경험 (Glassmorphism UI)
v8.1부터 적용된 **글래스모피즘(Glassmorphism)** 디자인은 지식 탐색의 즐거움을 더해줍니다. 모든 패널과 카드는 반투명한 유리 질감으로 처리되어 있으며, 부드러운 애니메이션과 함께 직관적인 피드백을 제공합니다.

## 📅 2. 주간 지식 복기 (Weekly Review)
단순한 기록을 넘어, 주간 단위로 자신의 지식을 되돌아보는 핵심 인터페이스입니다.
- **주간 선택기**: 검색창 하단 배지에서 원하는 날짜를 클릭하여 해당일의 활동을 즉시 확인하세요.
- **활동량 도트**: 날짜 아래의 작은 도트 색상(0~4단계)을 통해 그날 얼마나 많은 지식을 축적했는지 한눈에 알 수 있습니다.
- **데드라인 알림**: 마감일이 임박한 메모가 있는 날짜에는 특별한 알림 표시가 나타납니다.
- **모드 전환**: 📅 버튼을 통해 [주간 모아보기]와 [전체 목록]을 원클릭으로 전환할 수 있습니다.

## ✍️ 3. 고효율 지식 작성 및 연결

### 3.1 슬래시 명령어 (Slash Commands)
에디터 본문에서 `/`를 입력하면 다양한 서식과 기능을 즉시 호출할 수 있습니다.
- `/task`: 체크박스 생성
- `/ai`: AI 요약 요청
- `/h2`, `/h3`: 제목 서식 적용
- `/code`: 코드 블록 삽입

### 3.2 비주얼 노드 링커 (Alt + 드래그 와이어링)
지식과 지식을 선으로 잇는 가장 직관적인 방법입니다.
1.  메모 카드의 **#ID** 부분에 마우스를 올립니다.
2.  `Alt` 키를 누른 상태로 다른 메모 카드로 드래그합니다.
3.  화살표가 생기며 연결이 완료되고, 본문에 `[[#ID]]` 링크가 자동 삽입됩니다.

### 3.3 고속 편집 (Instant Edit)
메모 카드 위에 마우스를 올리고 **`e`** 키를 누르면 즉시 편집 모드로 진입합니다.

---

## ⌨️ 4. 단축키 워크플로우 (Navigation Shortcuts)
마우스 이동을 최소화하고 키보드만으로 지식 우주를 탐험하세요.
- **`Alt + 1~5`**: 사이드바의 주요 그룹(전체, 별표, 오늘, 파일, 완료)으로 즉시 이동합니다.
- **`Alt + 9`**: 설정 모달을 엽니다.
- **`n`**: 신규 메모 작성기를 엽니다.
- **`Esc`**: 모든 모달을 닫고 메인으로 돌아옵니다.

---

## ⚙️ 5. 고급 설정 및 보안

### 5.1 카테고리 시스템 (Advanced Mode)
설정에서 **"카테고리 기능 활성화"**를 체크하면 지식을 더 깊은 분류로 관리할 수 있습니다. 사이드바에 카테고리 필터가 추가되며, 작성 시에도 전용 카테고리를 지정할 수 있습니다.

### 5.2 이중 보안 암호화
메모마다 고유한 비밀번호로 본문을 잠글 수 있습니다. 암호화된 메모는 목록에서 내용이 보호되며, 오직 올바른 비밀번호를 입력했을 때만 해독되어 표시됩니다.

## 🚀 6. 지식 시각화 (Nebula & Relations)
- **지식 네뷸라**: D3 시뮬레이션을 통해 지식의 성단을 탐험하세요. 연결된 지식들은 서로 인력을 가집니다.
- **관계 포커스**: 특정 메모를 선택하면 그와 연결된 '맥락의 흐름'만을 하이라이트하여 복잡한 지식망을 단순하게 분석할 수 있습니다.
