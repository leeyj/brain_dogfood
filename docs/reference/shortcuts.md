# ⌨️ 단축키 가이드 (Keyboard Shortcuts) (v8.1+)

'뇌사료'를 더욱 빠르고 효율적으로 활용하기 위한 단축키 일람입니다. 
마우스를 최소화하고 키보드만으로 지식 우주를 고속으로 탐험하세요.

---

## 🚀 전역 네비게이션 (Global Navigation)
어느 화면에서나 즉시 실행되어 원하는 영역으로 이동합니다.

- **`Alt + 1` ~ `9`**: 🚀 **왼손의 법칙 전역 네비게이션**
    - `Alt + 1`: 🌐 전체 지식 (All Memos)
    - `Alt + 2`: ⭐ 별표 메모 (Starred)
    - `Alt + 3`: 📅 오늘의 메모 (Today)
    - `Alt + 4`: 🗓️ 이번주 메모 (Weekly)
    - `Alt + 5`: 📎 첨부파일 모음 (Files)
    - `Alt + 6`: ✅ 완료 및 보관 (Archive)
    - `Alt + 7`: 🔍 지식 탐색기 (Explorer)
    - `Alt + 8`: 🕸️ 지식 네뷸라 (Graph View)
    - `Alt + 9`: ⚙️ 환경 설정 (Settings)
- **`Alt + `** (Backtick): ⚡ **Quake 스타일 새 메모** (작성기 즉시 호출)
- **`Alt + W`**: 📅 **주간 뷰 표시 토글** (Visibility Toggle)
- **`Esc`**: 🚪 **모든 모달 닫기** 및 메인 화면 복귀
- **`?`**: ⌨️ **단축키 가이드 열기**

---

## 📝 메모 카드 조작 (Card Interaction)
메모 카드 위에 마우스를 올린 상태에서 실행됩니다.

- **`e` (Key)**: ⚡ **즉시 수정 (Instant Edit)** - 모달 없이 바로 편집 모드 진입
- **`s` (Key)**: ⭐ **별표(Pinned) 토글**
- **`d` (Key)**: ✅ **완료(Done) 처리**
- **`#ID` 클릭**: 📋 **내부 링크 복사** (`[[#ID]]`)
- **`Alt + #ID` 클릭 후 드래그**: 🔗 **비주얼 노드 링커** (시각적 와이어링 시작)

---

## ✍️ 메모 편집 (Editor Mode)
작성기(Composer) 및 수정 모달 내에서 사용됩니다.

- **`Ctrl + Enter`** 또는 **`Ctrl + S`**: 💾 **저장 및 게시**
- **`/` (Slash)**: 🪄 **슬래시 명령어 메뉴 호출**
- **`Ctrl + Alt + 1 ~ 4`**: 🏷️ **카테고리 칩 즉시 지정**
- **`Shift + Esc`**: 🗑️ **작성 취소 및 닫기**

---

> [!TIP]
> **`Alt + 1~9`** 단축키는 왼손만으로 주요 메뉴를 전환할 수 있도록 설계되었습니다. 마우스는 지식 간의 연결(Wiring)이나 복잡한 그래프 탐색 시에만 보조적으로 사용하면 작업 효율이 극대화됩니다.
